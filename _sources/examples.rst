Examples
========

Brownian motion
----------------

.. code-block:: bash

    python examples/brownian_motion.py

Path graph wiggle
-----------------

.. code-block:: bash

    python examples/path_graph_wiggle.py

Blender cones with per-frame color/size/orientation and edges
-------------------------------------------------------------

.. code-block:: bash

    python examples/blender_3d_cones_graph_demo.py

Notes
-----

- Matplotlib examples can save MP4s if ``ffmpeg`` is installed (see ``examples/output`` for samples).
- Blender examples require Blender installed; render path can be set via ``save_path`` in the config.
